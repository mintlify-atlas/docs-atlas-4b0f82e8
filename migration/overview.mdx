## What version should I use?

Always use the latest stable version of Geode when developing new mods. Migration guides help you update existing mods when new major versions are released.

<Note>
If you're starting a new mod, skip these guides and follow the [Getting Started](/getting-started/prerequisites) tutorial instead.
</Note>
